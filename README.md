# 上财保障菜配餐助手

这是一个用于《人工智能模型应用实践》结课实验的本地 Web 应用。作品以“上海财经大学学生食堂保障菜价格信息（2023-2024学年）”公开页面为菜名和售价来源，通过本地规则完成预算过滤、忌口过滤、营养估算、套餐评分和结果解释。

项目不调用真实大模型，也不依赖外部 API。页面运行时只加载本项目内的 HTML、CSS、JavaScript 和本地图片资源。

## 数据来源

- 公开来源：上海财经大学信息公开网
- 页面标题：学生食堂保障菜价格信息（2023-2024学年）
- 来源链接：https://gongkai.sufe.edu.cn/a6/f3/c13786a42739/page.htm
- 使用范围：菜品名称、菜品类别和公开售价
- 本地估算：热量、蛋白质、标签、综合评分和推荐理由

## 功能

- 按轻盈控卡、高蛋白、省钱吃饱、均衡能量生成套餐。
- 根据预算、口味和“不吃辣”过滤菜品。
- 展示主菜、配菜、主食/点心组合及推荐理由。
- 从高分候选套餐中随机选择，避免同样条件每次完全相同。
- 使用浏览器 `localStorage` 保存偏好、收藏套餐、收藏菜品和反馈。
- 展示本地数据库状态、公开来源和最近保存记录。
- 使用 16 张真实菜品照片和 3 张上财公开报道现场照片，现场图只用于场景说明，菜品卡片使用菜品照片。

## 运行

直接打开：

```text
campus_ai_helper/index.html
```

或使用本地静态服务访问：

```powershell
node server.cjs 8766 127.0.0.1
```

访问：

```text
http://127.0.0.1:8766/index.html
```

如果要在同一局域网内的其他设备访问，可以绑定到全部网卡：

```powershell
node server.cjs 8767 0.0.0.0
```

## 公网展示

本项目是纯静态页面，不需要服务端数据库。可部署到 GitHub Pages、Vercel、Netlify，也可用 `cloudflared tunnel --url http://127.0.0.1:8766` 生成临时公网链接。

访问者的收藏、反馈和历史记录保存在访问者自己浏览器的 `localStorage` 中，不会写入服务器。

## 测试

```powershell
npm test
node --check src/mealCore.js
node --check src/app.js
node --check server.cjs
```

## 文件说明

- `index.html`：应用入口页面。
- `styles.css`：页面样式。
- `src/mealCore.js`：公开菜价数据、本地筛选、评分和数据库配置说明。
- `src/app.js`：页面交互、本地数据读写、收藏和反馈。
- `assets/`：本地菜品照片、上财公开现场照片和图片来源记录。
- `tests/`：核心逻辑、页面文案和交互测试。
- `结课实验作品说明.md`：课程提交说明。
- `final_report.docx`：正式 Word 报告。
